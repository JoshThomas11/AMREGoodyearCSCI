ImageJ 1.44

This is the full 1.44 release of ImageJ. You can easily upgrade to newer letter releases (e.g., 1.45e) using Help>Update ImageJ command. The Image User Guide (8MB PDF) has a list of the new features in this release. The ImageJ website is at rsb.info.nih.gov/ij/ and the OS X installation notes are at rsb.info.nih.gov/ij/docs/install/osx.html. This release includes both 32-bit ("ImageJ") and 64-bit ("ImageJ64") applications. "ImageJ64" requires a 64-bit Intel processor running OS X 10.5 or later. The 32-bit version ("ImageJ") is required to run plugins and commands (e.g., File>Import As>Using QuickTIme) that use QuickTime.
